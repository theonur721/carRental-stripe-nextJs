# 🚗 Car Rental

Kullanıcıların araçları görüntüleyip sipariş edebildiği, Stripe ile güvenli ödeme sistemi entegre edilmiş modern bir **araç kiralama uygulaması**.

---

## Kullanılan Teknolojiler

- **Next.js 15** – App Router & API Routes
- **React 19** – Modern kullanıcı arayüzü
- **MongoDB + Mongoose** – Veritabanı yönetimi
- **Stripe API** – Güvenli ödeme altyapısı
- **Tailwind CSS 4** – Responsive ve modern tasarım
- **React Icons** – İkon seti
- **Millify** – Sayı formatlama

---

## Özellikler

- 🏎️ Araçları listeleme ve detay görüntüleme
- 💳 Stripe ile güvenli ödeme
- ✅ Ödeme onay ve başarısız ekranları
- 📋 Siparişlerim sayfasında kiralanan araçları listeleme
- 📱 Tüm cihazlarda tam responsive tasarım
- 🎨 Responsive , modern ve sade kullanıcı arayüzü
